# SQL Portfolio

This section contains my SQL projects developed using the MySQL Sakila sample database.

The objective of these projects is to strengthen my SQL skills through practical business-oriented exercises covering data extraction, filtering, joins, aggregations, subqueries and analytical queries.

## Technologies

- MySQL
- MySQL Workbench
- Sakila Database

## Topics Covered

- SELECT
- WHERE
- ORDER BY
- GROUP BY
- HAVING
- Aggregate Functions
- JOINs
- Subqueries
- Common Table Expressions (CTEs)
- Window Functions

The projects are organized by increasing difficulty to reflect my learning progression.
